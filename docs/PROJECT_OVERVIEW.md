# 项目总览

## 项目名称

租房管理微信小程序原型工程

## 项目目标

为二房东、职业房东提供一套适合微信小程序端使用的自我管理工具，覆盖以下闭环：

1. 房源录入与结构管理
2. 空置房办理入住
3. 按租期与支付周期生成账单
4. 水电抄表与杂费生成
5. 房租 / 水电 / 灵活收费记账
6. 退租与历史留痕
7. 档案管理与报表导出

## 当前阶段

当前阶段是：

- 高保真原型复刻
- 微信小程序交互验证
- 使用本地 mock 数据进行状态驱动
- 逐步为未来的后端落库做结构准备

尚未完成：

- 真实后端 API
- MySQL / Prisma 落库
- Cloudflare R2 真实上传链路
- JWT 登录鉴权
- 真正的 Excel 导出

## 主要页面

- `工作台`
- `楼栋详情`
- `房间详情`
- `办理入住`
- `账务`
- `我的`

## 重要说明

- 目标端是微信小程序，不是 PC Web。
- H5 仅用于辅助调试。
- 样式设计优先贴近 `rent.txt` 原型。
- 当前项目允许逐页迭代、逐页校对，不要求一次做完。
