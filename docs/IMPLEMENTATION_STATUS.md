## 3. 已知问题

### 高优先级

- 部分页面仍有中文乱码或问号占位，尤其是 `我的` 页和部分 mock 数据文案
- 部分文件存在编码污染，需要系统性清理
- 某些交互虽然可用，但和原型仍有差距

### 中优先级

- `账务`、`我的` 视觉层级还可继续逼近原型
- 结构管理页仍需继续精修
- 个别抽屉页还需要继续减小顶部留白并优化滚动体验

### 技术债

- `wx.getSystemInfoSync` 仍在使用，微信开发者工具会给出弃用警告
- 构建链路依赖脚本清理 WXSS，不够优雅
- 数据目前是本地持久化，不是正式后端落库

## 4. 当前推荐接手顺序

新接手的人不要直接从零看代码，按这个顺序进入：

1. 阅读 `docs/PROJECT_OVERVIEW.md`
2. 阅读 `docs/PRODUCT_REQUIREMENTS.md`
3. 阅读 `docs/TECH_ARCHITECTURE.md`
4. 阅读 `docs/PROJECT_HISTORY.md`
5. 再进入这些代码文件：
   - `src/data/rentStore.js`
   - `src/pages/workbench/index.vue`
   - `src/pages/room/detail.vue`
   - `src/pages/bills/index.vue`
   - `src/pages/profile/index.vue`
